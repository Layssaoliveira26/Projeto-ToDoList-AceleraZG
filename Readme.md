# Projeto To Do List
O presente projeto, construído ao longo do programa Acelera ZG tem comoobjetivo realizar operações de cadastro de atividades, consulta de atividades e exclusão.

## Tecnologias utilizadas
- Java: construção da lógica do backend;
- Intellij: ambiente de desenvolvimento utilizado para organização de código, testes e execução.

## Execução do projeto
- shift + f10 ou execução do método main presente na classe Main;
